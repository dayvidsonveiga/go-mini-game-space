# Jogo 2D em Go

<h2>Descricao</h2>
Esse jogo foi criado utilizando a lib Ebiten Engine

----------

<h2>Como rodar</h2>
1. Clone o repositório

```
git clone
```

2. Entre no diretório e rode o seguinte comando para instalar as dependencias

```
go mod tidy
```

3. Inicie o jogo

```
go run main.go
```
